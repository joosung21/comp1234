Assignment2 Reflection

The Tech Conference 2024 landing page project allowed me to apply HTML and CSS in a way that created a clear and organized structure. By using semantic HTML tags for sections like the header, navigation, main content, and footer, I was able to improve both readability and accessibility. This helped ensure the page was structured in a way that supports users and also makes the code itself easier to maintain.

Typography and color schemes were essential in creating a cohesive visual identity. I used Google Fonts, combining Roboto Slab and Roboto Mono for enhanced readability, while Material Icons added intuitive interactivity to elements like the "Back to Top" button. The color scheme, inspired by the George Brown College logo, used blue and yellow tones, lending a sense of consistency across the page and reinforcing the theme of the conference.

Overall, this project offered valuable experience in implementing visual cohesion, typography, and interactivity to elevate user experience. It deepened my understanding of how different design elements can come together to form a functional, visually appealing, and accessible webpage.
