Assignment2 Reflection

Designing the Tech Conference 2024 landing page allowed me to apply various HTML and CSS skills, focusing on front-end development and UI/UX design. I structured the page into clear sections, using semantic HTML tags to enhance accessibility and readability. Implementing responsive design was a key challenge. I used media queries to ensure the layout adjusted properly on different screen sizes, making the page functional and visually appealing on both desktop and other smaller devices.

I also explored visual design elements, using Google Fonts like Roboto Slab and Roboto Mono for clear typography, and integrated Material Icons for interactivity, such as the "Back to Top" button. The color scheme was inspired by the George Brown College logo, and I applied the blue and yellow tones throughout the design to create consistency.

In conclusion, this project helped me strengthen my understanding of responsive web design, typography, and how to create a cohesive user experience. It was a valuable opportunity to blend aesthetics with functionality in a real-world scenario.
